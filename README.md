# Study Support App
Hệ thống hỗ trợ học tập cá nhân giúp quản lý môn học, bài tập và tài liệu.

## Các tính năng chính
- Đăng nhập/Đăng ký tài khoản.
- Quản lý thông tin cá nhân.
- Quản lý danh sách môn học.
- Theo dõi tiến độ bài tập.
- Lưu trữ và quản lý tài liệu học tập.

## Công nghệ sử dụng
- Java (MVC Pattern)
- JSP/Servlet
- Bootstrap 5
- Git/GitHub
